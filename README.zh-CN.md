# 图片本地化 Image Localization

Image Localization 是一个 Codex Skill，用于把源图片素材处理成可直接投放的多语言广告和社交平台素材。

它优先使用 Codex 内置的视觉理解、图像生成和图像编辑能力，追求原生视觉效果。代码只负责确定性的辅助工作，例如尺寸规范化、等比裁切、命名、manifest 和 QA 总览图。

## 核心卖点

这个 Skill 使用 Codex 内置的图像能力，因此可以直接消耗 Codex 订阅额度来生成和本地化图片。不需要额外配置图片 API，不需要单独准备 API Key，也不需要接入新的计费服务。

它的缺点是速度相对慢：Codex 内置图像流程不如专门的批量图片 API 快。但对市场、投放、运营同学来说，它依然很适合长期跑任务，因为配置成本低、原生视觉效果好、综合性价比高。

未来我会更新另一个基于 Nano Banana API 的 Skill，用于更快的大批量产出。

## 能做什么

- 翻译图片中的可见文字。
- 保留品牌名、产品名、Logo、主体和视觉层级。
- 输出常见广告/社交尺寸：
  - `1200x1200`
  - `1920x1080`
  - `1080x1350`
  - `1080x1920`
  - `1200x628`
- 对特殊分辨率，先生成最接近且安全的常见比例，再用确定性等比裁切处理。
- 维护品牌/产品术语记忆。
- 输出规范命名的文件和 manifest。
- 交付前进行视觉 QA。

## 设计原则

目标不是简单蒙版换字，也不是机械拉伸图片，而是生成看起来原生、自然、适合投放的本地化素材。

职责划分：

- 模型负责：识别、翻译、图像编辑、版式重排、画布扩展、视觉修复。
- 代码负责：精确缩放、等比裁切、文件命名、manifest、contact sheet、尺寸检查。

默认不使用模糊背景填充。

## 尺寸处理

以 `1200x628` 为例，推荐流程是：

1. 先让模型生成带上下安全边的 `16:9` 版本。
2. 将 `1920x1080` 等比缩放到 `1200x675`。
3. 再裁掉上下多余像素，得到 `1200x628`。

这样可以保持几何比例，避免非等比压缩导致文字或主体变形。

## 术语记忆

Skill 支持用 JSON 文件保存用户确认过的术语规则：

```json
{
  "version": 1,
  "brands": {
    "example-brand": {
      "display_name": "Example Brand",
      "rules": [
        {"term": "Example Brand", "action": "preserve"}
      ],
      "products": {}
    }
  }
}
```

品牌级规则适用于该品牌下的所有产品。产品级规则可以覆盖品牌级规则。
